Industrial Strength Connection Pooling for OpenOffice

OOPool is a production ready connection pool for open office which manages starting and stopping instances of OpenOffice to dynamically respond to a changing load while ensuring that the system does not suffer from resource exhaustion.

Features

Cross-platform
OOPool runs equally well on Windows®, Mac OS X and Linux and is capable of spawning either LibreOffice or OpenOffice instances.
Clustered
OOPool can distribute requests to multiple office instances running on a single server or to multiple OpenOffice instances running on multiple servers in a clustered architecture.
Small foot-print
OOPool will stop OpenOffice instances which are not being used, minimizing the use of scarse computing resources.
Prevent Memory Leaks
OOPool is capable of stopping and restarting OpenOffice instances on a heavily loaded server to prevent memory-leaks from becoming a problem.
Metrics
OOPool generates metrics allowing you to visualize resource consumption and aide with planning.

Why use connection pooling?

OpenOffice & LibreOffice both provide fantastic support for managing and manipulating documents through their API allowing developers to establish a connection to a running OpenOffice instance and leverage its mature document conversion and rendering technologies to produce stunning documents in a variety of formats.

Unfortunately a single instance of OpenOffice can only process one document at a time which can result in throughput bottlenecks while some requests are blocked or rejected until the instance has finished processing all of the previous requests. Additionally OpenOffice when running for long periods of time or under heavy load is known to suffer from memory leaks which require an operator to manually stop and restart the instance. This is an inconvenience when there are a small number of infrequent requests but is a major burden in environments under heavy load.

If you need to using the API of OpenOffice in a large or highly available deployment where downtime isn't accepted OOPool will make your life better.
